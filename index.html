Schritt 2:

Kopiere den unten stehenden Code jeweils in ein Lesezeichen. Der erste Code ist für die Buchung von gewöhnlichen Kursen, wobei keine Kontoverbindung ausgefüllt werden muss. Der zweite Code ist für die Buchung der Sportscard, die eine Kontoverbindung benötigt. (Hinweis: Aktuell funktioniert nur die Option "Studierende/r der BU Wuppertal" korrekt, obwohl die anderen Optionen wählbar sind.)

Schritt 3:

Öffne die Buchungsseite des Unisports und wähle den Kurs bzw. die Sportscard zum Buchen aus. Wenn du aufgefordert wirst, deine Daten einzugeben, klicke auf das entsprechende erstellte Lesezeichen (Kurs bzw. Sportscard) und die Formulare füllen sich. Ab dort einfach die Buchung wie gewohnt fortsetzen.
